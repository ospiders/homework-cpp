общие значения для A, B, C;
■ Создайте одномерный массив, который содержит 
уникальные значения для A, B, C;
■ Создайте одномерный массив, который содержит 
общие значения для A и C;
■ Создайте одномерный массив, который содержит 
отрицательные значения для A, B, C без повторений.
